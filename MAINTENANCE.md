# 🛠️ 知識花園維護指南 (Knowledge Garden Maintenance)

這份文件紀錄了如何維護與檢查知識花園的運行狀態。

## 🎨 外觀修改
若需調整網站的配色或佈局，請直接編輯 `index.html` 中的 `<style>` 區塊。

- **主要色調 (`--garden-green`)**: 控制整體綠色基調。
- **強調色 (`--accent-pink`)**: 控制點綴與強調部分的顏色。

## ✅ 維護檢查清單
- [ ] **JSON 格式檢查**：確保 `knowledge.json` 的逗號與括號正確，避免導致網頁崩潰。
- [ ] **ID 唯一性**：確保每個新條目的 `id` 是唯一的且按順序遞增。
- [ ] **Git 同步狀態**：確認本地變更已成功 `push` 至遠端倉庫。
- [ ] **路徑正確性**：確認所有內部連結或引用路徑在更新後依然有效。
