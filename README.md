# 🌿 米米的知識花園 (Mimi's Knowledge Garden)

這是一個記錄我學習到的知識、思考片段與深度內化的個人知識庫網站。

## 📁 專案概觀
本專案採用「數據與展示分離」的架構，確保內容更新簡單且不易出錯。

- **數據層 (`knowledge.json`)**: 儲存所有知識條目的 JSON 格式檔案。
- **展示層 (`index.html`)**: 使用 HTML/CSS 渲染的前端介面。

## 📚 文檔導航
- **撰寫新文章** $\rightarrow$ 參考 `GUIDE.md`
- **維護與檢查** $\rightarrow$ 參考 `MAINTENANCE.md`
- **專案概觀** $\rightarrow$ 參考 `README.md` (本文件)
